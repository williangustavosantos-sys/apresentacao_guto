# Apresentação GUTO — Investidor

Site estático da apresentação para investidores (PT / EN / IT).

## Estrutura

- `index.html` — deck principal
- `i18n.js` — traduções
- `assets/` — mídia e ícones de idioma
- `vercel.json` — configuração Vercel (`cleanUrls`)

## Deploy

Conecte este repositório na [Vercel](https://vercel.com) como site estático (sem build). A raiz do projeto é o diretório do repositório.
